## **entity_verb各个python文件简介**
* entity_verb_new.py:判断文档的每句话中，包含有哪些xlink和bdlink命中的实体，
若该句话命中实体，则用TLP对其进行词性标注，再用thulac对这句话分词，
进而将动词提取出来。因为每句话中同一实体可能出现多次，所以在实体名称
前加上序号，表明这是这句话第几个该实体。输出：entity_verb_result / file_name.txt
~~~~
[['1#北京故宫博物院_ns', 0], ['1#北京_ns', 0], ['1#北京故宫_ns', 0], ['1#故宫_ns', 2], ['建立_v', 7], ['位于_v', 22], ['2#北京_ns', 24], ['2#北京故宫_ns', 24], ['2#故宫_ns', 26], ['1#紫禁城_n', 28]]
~~~~
